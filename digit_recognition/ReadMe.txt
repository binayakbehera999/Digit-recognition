========================================================================
    CONSOLE APPLICATION : 224101014_Digit Recognition Project Overview
========================================================================

This is project to detect if the speech wave is a digit (0-9).

Device Model Name - ASUS-ROG-GL-503
The DC shift is calculated using device michrophone.

Instructions to execute the code:
1. Press F5
2. Enter a choice.


Description of the Code:
1. Training is done on 20 files.
2. Pre-recorded testing is done on 10 files.
